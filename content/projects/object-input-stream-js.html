ObjectInputStream JS

Java, like most languages, allows you to serialize ("pickle") objects. Its serialization protocol is called the Object Stream Protocol. To read it you'd use the java.io.ObjectOutputStream class, and to write it - java.io.ObjectInputStream.

Over the years, several attempts have been made to deserialize that format using JavaScript, but all of them were incomplete, until this one.
